# YOUTUBE-VEDIOS-TO-HD-SCREEN-SHOTS-PDF-
This project is a Python-based tool that extracts high-quality (HD) screenshots from YouTube videos and converts them into a single PDF file.  It allows users to download a video, capture frames at specific intervals, and save them as images. These images are then combined into a well-structured PDF document.
This tool is useful for:

📚 Students (notes from lectures)
🎓 Teachers (presentation material)
🎥 Content creators (video analysis)
📄 Documentation purposes
⚙️ Features
🎬 Download YouTube videos
🖼️ Extract HD screenshots at custom intervals
📁 Save images locally
📄 Convert screenshots into a PDF
⚡ Simple and easy-to-use Python script
🛠️ Technologies Used
Python
OpenCV (cv2) – for frame extraction
pytube / yt-dlp – for video download
PIL (Pillow) – for image processing
fpdf / reportlab – for PDF generation
🚀 How It Works
Enter YouTube video URL
Download the video
Extract frames at specified time intervals
Save frames as HD images
Combine all images into a PDF
